# 📂 โน้ตบุ๊กศูนย์รวมข้อมูลระบบ (Project Context & Docs)

ยินดีต้นรับสู่ **"สมองส่วนกลาง"** ของโปรเจกต์ โฟลเดอร์นี้สร้างขึ้นมาเพื่อเก็บรวบรวม **Context, ข้อมูลดิบ, แผนงาน และกฎเกณฑ์ทั้งหมด** ก่อนที่เราจะเริ่มลงมือเขียนโค้ดจริง (Day 0) 

เป้าหมายสูงสุดคือเพื่อให้เราทั้งสองคน มองเห็นภาพสถาปัตยกรรมแอปพลิเคชันตรงกัน 100% โดยไม่ต้องเดา

> ⚠️ **กฎเหล็กประจำเป็นแผนก:** 
> 1. ห้ามลบไฟล์ในโฟลเดอร์นี้เด็ดขาด ถ้าต้องการเปลี่ยนแปลงเชิงโครงสร้าง รบกวนเปิดประเด็นคุยกันในแชตก่อนตลอด
> 2. ทุกครั้งที่อัปเดตระบบ หรือปรับเปลี่ยนแผนงาน ให้มาบันทึกที่ **Activity Log** ด้านล่างสุดของไฟล์นี้ด้วย
